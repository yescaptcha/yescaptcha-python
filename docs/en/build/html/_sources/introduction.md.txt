# Introduction

This document is used to introduce how to use the Python SDK - yescaptcha-python of the [YesCaptcha](https://yescaptcha.com/) service.

The SDK provides the encapsulation of all service interfaces of YesCaptcha, so that we can use YesCaptcha related services more conveniently.

The specific contents of this document include:

- installation method
- Basic initialization use
- Use of each task
